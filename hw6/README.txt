The program contain 4 sorting. Selection, Bubble, Insert, and Quick sort.
When executing the program, there are 4 options. Enter 1 to use selection
sort. Enter 2 to use bubble sort. Enter 3 to use insert sort. Enter 4 to 
use quick sort.

function:
	void selection_sort(int *arr, int tot)
		
	void bubble_sort(int *arr, int tot)
	
	void insertion_sort(int *arr, int tot)
	
	void quick_sort(int *arr, int left, int right, int tot)    
	Notice: tot is just for printing the result.
