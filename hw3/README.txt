The program is about the Josephus problem which is implemented by circular linked list and double linked list.

Firstly, you can enter the number to choose whether to input names by yourself or reading data from the file.
If you want to leave, press 0.
Afterwards, every name stored will be displayed on the screen. 

Scondly, choose either right or left, and enter the skip number to delete the name in the linked list.
The result is showed including the list lengths, and the name which is poped in diffferent moments.
After all the steps are done, finally, the last data will be listed at the bottom of the line.
